## Announcement

Making Playgrounds using Claude Code

We've published a new Claude Code plugin called **playground** that helps Claude generate HTML playgrounds.

## What Are Playgrounds?

**Standalone HTML files** that let you:
- Visualize a problem with Claude
- Interact with it
- Give you an output prompt to paste back into Claude Code

## Use Cases

"I've found this can be really good interacting with the model in ways that are not well suited for text, for example to:"

(Specific examples follow in the Favorite Playgrounds section)

## Getting Started

Install the plugin in Claude Code by running:

```bash
/plugin marketplace update claude-plugins-official
/plugin install playground@claude-plugins-official
```

### Interactive Feedback Loop
