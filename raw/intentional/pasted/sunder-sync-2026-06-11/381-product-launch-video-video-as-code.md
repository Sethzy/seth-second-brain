# Product Launch Video - Video-as-Code

## Idea

Create a polished product launch video using video-as-code tools. Key insight: **Remotion now has Agent Skills** - can generate videos directly with Claude Code prompts.

## Research: Video-as-Code Rankings (Jan 2026)

| Rank | Tool | Stars | Best For | Weakness |
|------|------|-------|----------|----------|
| 1 | **Manim** | ~65k | Math/logic animations, "genius" aesthetic | Python-based, hard to use React components |
| 2 | **Remotion** | ~21k | React ecosystem, NPM packages, Tailwind | Slowest render (headless browser) |
| 3 | **Motion Canvas** | ~18k | Built-in editor, real-time preview | Canvas-only, no HTML/CSS layouts |
| 4 | **Theatre.js** | ~12k | Cinematic 3D/Three.js keyframing | Not a renderer, needs separate MP4 tool |
| 5 | **Editly** | ~5.5k | Simple clip stitching | Lacks fine-grained animation |
| 6 | **Revideo** | ~4k | Fast server-side rendering, APIs | Younger ecosystem |
## Recommendation

**Motion Canvas** for "most awesome" result with least pain - pro editor feel with TypeScript source of truth.

**BUT: Remotion with Agent Skills** is the game-changer for speed:

```bash
npx skills add remotion-dev/skills
```

Generate videos just by prompting Claude Code. Example animation created purely via prompts - terminal typewriter effect, 3D transforms, logo reveals.
