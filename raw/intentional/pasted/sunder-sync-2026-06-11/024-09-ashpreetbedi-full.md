## Technical Architecture

### Context Layers Explained

1. **Table usage** - How tables are accessed and used historically
2. **Human annotations** - Expert knowledge and metadata
3. **Query patterns** - Common queries and access patterns
4. **Institutional knowledge** - Organizational context and semantics
5. **Memory** - Persistent agent memory across sessions
6. **Runtime context** - Current session and task context
