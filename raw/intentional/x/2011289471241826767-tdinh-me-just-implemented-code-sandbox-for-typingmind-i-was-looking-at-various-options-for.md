Just implemented Code Sandbox for TypingMind 🥳

I was looking at various options for sandbox and code execution, but for now the most obvious choice is to use the built-in sandbox provided by all popular providers: OpenAI/Gemini/Claude.

It can create files but not yet able to accept file input, except for OpenAI's API, which gives your 20 minutes of "persistent container" which persist the files and the execution session.

I'll look into implement 3rd party provided sandboxes like Vercel sandbox or E2B soon!
