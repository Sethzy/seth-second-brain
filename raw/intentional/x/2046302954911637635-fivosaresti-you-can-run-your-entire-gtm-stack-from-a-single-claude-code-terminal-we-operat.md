## Verbatim Text

You can run your entire GTM stack from a single Claude Code terminal.

We operate 22 apps with zero context switching.

Claude Code has become our company OS.

Here is what each one does:

1. Core ops
• Google Workspace: docs, sheets, drive management
• GitHub: repos, PRs, issue management
• 1Password: agents get credential access
• Notion: search and update our internal wiki
• Slack: draft replies and manage DMs

2. Sales & outreach
• Apollo: source and enrich accounts
• Clay: source and enrich contacts
• Findymail: source missing emails and verify them
• InstantlyAI: spin up email campaigns and pull reports
• HeyReach: automate DM prospecting campaigns
• HubSpot: manage objects, query pipeline

3. Research & data
• Firecrawl: search the web and scrape pages
• Pinecone: store data in our vector databases
• Exa: AI-powered web search
• Browserbase: control the browser from the terminal

4. Design & content
• Figma: pull designs and design context into code
• Miro: build workflow diagrams and processes
• beehiiv: manage our newsletter

5. Finance & infra
• Stripe: check payments and manage invoices
• Airtable: read and write to our databases
• Railway: deploy and manage apps
• Gmail: search past email threads for context
• Google Calendar: check availability and book meetings

Our day-to-day now starts in the terminal.

The best software will add new capabilities to AI agents.
