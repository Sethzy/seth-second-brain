OpenAI announced two new deep research API models, webhooks for API event notifications, web search availability for o3/o3-pro/o4-mini models, reduced web search pricing, and OpenAI DevDay 2025

- o3-deep-research-2025-06-26 ($10/$40 per 1M input/output tokens) described as most powerful deep research model with highest reasoning but slowest speed, and o4-mini-deep-research-2025-06-26 ($2/$8 per 1M input/output tokens) as faster, more affordable deep research model with higher reasoning and medium speed

- Both models are the same post-trained o3 and o4-mini models that power deep research in ChatGPT, supporting MCP (search/fetch), Code Interpreter, designed to handle complex multi-step research tasks, and can search and synthesize information from across the internet as well as from your own data brought in through MCP connectors

- Web search is now available with o3, o3-pro, and o4-mini models that can search the web within their chain-of-thought, with search content tokens billed at model rate $10/1k calls plus input token rates for o3/o4-mini models, and reduced pricing to $25/1k tool calls for 4o and 4.1 models across the board

- Added webhooks for async event handling to receive notifications for certain API events such as completed responses, fine-tuning jobs, and batch jobs instead of polling, recommended for long-horizon tasks like deep research to improve reliability

- OpenAI DevDay on October 6, 2025 in San Francisco described as the biggest one yet with 1500+ developers, livestreamed opening keynote, hands-on building with their latest models and tools, more stages and more demos
