LLM Pricing Tiers 💰

I just updated my pricing table.

The table now indicates 3 pricing tiers:

• tier 1 is $1 to $7
• tier 2 is $12 to $24
• tier 3 is $42 to $120

To calculate "total cost", I combined input cost and output cost.

I use a 3:1 ratio, assuming there are 3 input tokens for every 1 output token.

This ratio is my own creation and not backed by science - YMMV 🙂

Tier 1 models are very fast and cheap. Haiku is the star in this group. In RAG, I use tier 1 models for quick tasks that have slight room for error, like query transformation.

Tier 2 models are slower, but yield higher complexity. Command R+ has been my choice here. In RAG, I use tier 2 models for fast, but more critical tasks like function calling.

Tier 3 models are the crème de la crème. Opus is the clear stand out for me.  For coding, I only use Opus. In RAG, I use tier 3 models as the output layer LLM.
