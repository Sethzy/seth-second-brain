## Verbatim Text

helpful sales framework we used during YC to make sure the next steps are clear after each sales demo:

1) “does next Thursday at 2pm work to review this?”
If they respond with “I’ll reach out once I know”, follow up with:

2) “totally fine. Let’s put a placeholder on the calendar for now. It’s much easier to move it later than trying to chase each other next week”
If they are not fine with that, try:

3) “fair enough. Would it help if we opened a shared Slack channel? I can drop everything there so it doesn’t get buried in email”
If they prefer email:

4) “understood. I’ll send a short summary today and follow up on Tuesday if I haven’t heard back. Does that work for you?”

always define the next steps before ending any sales call
