We generated 110 leads in a single week with $0 spent on ads.

3 things that made this outbound campaign work:

(1) We scraped data that competitors don’t have access to

Elena (our GTM engineer) built a custom Python scraper for public directories and got full business info + validated emails.

(2) AI personalization that sounds like a human wrote it

Lorenzo (our partner) built an AI agent to name-drop local competitors + a nearby coffee shop to meet at.

So every single message was unique and personalized.

(3) Landing in the primary tab

We started putting every mailbox in Instantly AI's premium warmup pool.

Now sending 1200 emails per day using it (our results have never been better).

___

I don’t see a reason why any company that is trying to grow shouldn’t be running outbound.

Even if you don’t want to hire an outbound sales team, you should at least have an automated outbound engine running.

Because cold outbound is still one of the most cost-effective B2B channels out there.
