## Verbatim Text

I hosted a Claude Code meetup today in Austin to talk about Agents, Skills, OpenClaw and context management

Here’s the agenda if you want to do one yourself:  

9:00A — start. Yes that’s early! But I found more serious people come if you do this during work hours

9:10A — introductions. Quick go-around for people to say their name and what they’re working on

9:20A — small groups. Split into groups of 3-4 to talk more deeply about what you’re building and how you use Agents and Claude Code etc 

9:30A — mix and mingle. I bought breakfast tacos ($350) and our venue sponsor let us drink their coffee 

9:45A — discussions on topics:

Agents
Claude for Marketers
OpenClaw
Context Management

Everyone picked one of these groups and went for demos and questions 

10:15A — mix and mingle. Good meetups have time for both structured and unstructured convos!

10:30A — demos in the classroom, this is where I took the photo. We got share-outs from each of the breakout sessions

Then people presented on:

OpenClaw and how to run it
Using Codex for checking code
NanoBanana API use

And a bunch more.

We were done by 11:30A and some folks stuck around for another 20-30 mins.

Overall it was one of my best meetups on a technical subject. I got a lot of great feedback

35+ Founders and Engineers and Entrepreneurs attended

If you applied and I wasn’t able to reply I’m sorry! I’ll try to host another one in Austin in a few weeks
